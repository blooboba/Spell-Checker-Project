# Java Spell Checker

A simple spell checker program built in Java using a custom Hash Table implementation using separate chaining.

## Features
- Loads a dictionary of 61,000+ English words
- Checks user input and reports whether each word is spelled correctly
- Uses a custom hash table with separate chaining
- Fast lookup performance

## How to Run
Compile:
    javac HashTable.java SpellChecker.java

Run:
    java SpellChecker

## Files
- HashTable.java — custom hash table implementation
- SpellChecker.java — main program
- WordList.txt — dictionary file
